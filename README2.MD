# Simple Scheduler

A Python script that schedules a task reminder after a delay.

## Description
This project allows users to enter a task and a delay time.
After the delay, it prints a reminder.

## Features
- Task scheduling
- Time delay
- Simple CLI interface

## Technologies Used
- Python
- time module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python scheduler.py

4. Enter task and delay time

## Example

Input:
Drink water  
5  

Output:
Reminder: Drink water
